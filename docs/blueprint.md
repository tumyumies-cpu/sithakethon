# **App Name**: PlatePal

## Core Features:

- Surplus Food Posting: Enable restaurants and hostels to post surplus food with details like type, weight, best before, and photos. Uses geolocation for pickup location.
- NGO Food Discovery & Reservation: Allow NGOs to browse available food offers, filter by criteria, and reserve items, with a cart feature to combine offers from multiple providers.
- Driver Task Management: Provide a mobile-friendly interface for drivers to view available tasks, accept assignments, and mark pickups and deliveries.
- Smart Matching: AI prioritizes urgent offers and suggests nearby NGOs and drivers, optimizing food distribution.
- Token Incentive System: Implement a token system for providers, NGOs, and drivers, incentivizing food donation, pickup, and distribution.
- Impact Dashboard: Display live metrics on food waste reduction, meals served, and partners involved, showcasing the platform's impact.
- Admin Verification: Allow administrators to verify organizations, resolve disputes, and oversee platform activity.

## Style Guidelines:

- Primary color: Deep teal (#0A8A72) evokes nature and sustainability.
- Background color: Very light gray (#F7FAFC) creates a clean, trustworthy foundation.
- Accent color: Sunny yellow (#FFD166) is used to highlight calls to action and important elements.
- Headings: 'Space Grotesk', sans-serif, providing a modern and trustworthy feel for headlines.
- Body: 'Inter', sans-serif, is used to provide readable, neutral, and objective text.
- Use consistent, easily recognizable icons for food types, user roles, and actions throughout the platform.
- Implement a mobile-first, responsive design to ensure accessibility on all devices. Use a card-based grid layout for features and offers.
